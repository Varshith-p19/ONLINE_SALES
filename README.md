# ONLINE_SALES
The objective of this analysis is to understand sales performance by calculating revenue, order volume, and identifying trends across different time periods such as year and month.


Online Sales Analysis (SQL Server)

This project analyzes an online sales dataset using SQL Server (SSMS).
The dataset contains **250 rows** with the following columns:

- `order_id`
- `order_date`
- `amount`
- `product_id`

The objective of this project is to perform:
- Data import
- KPI calculation (Revenue, Volume)
- Time-based analysis
- Sorting and filtering
- Grouping by year and month
